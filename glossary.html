Glossary¶

preprocessor directive
A line of C++ code that starts with a hashtag symbol # is processed by the preprocessor, and is then removed, so that it doesn’t appear in later phases of compilation.
boilerplate code
boilerplate
“Boilerplate code or boilerplate is the sections of code that have to be included in many places with little or no alteration.” [WPBoilerplate]
